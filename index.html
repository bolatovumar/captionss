Why captionss?

Uses nothing more than CSS3 and HTML5, no JavaScript needed.

No need to specify image sizes ahead of time, even works with dynamically sized images.

Lots of caption styles can be achieved with the simple set of options.

Check out the Usage section below for everything that captionss can do.

Getting Started

Adding captionss to your project

Add it with Bower...

bower install --save captionss

... or grab the source directly from the captionss GitHub repository.

Referencing captionss in your HTML file

Add a link tag like the following referencing the location of captionss.

<link href="bower_components/captionss/captionss.min.css" rel="stylesheet" type="text/css">

Adding a caption to an image

Wrap the image in a figure tag and then add a figcaption tag below it with the caption text.

<figure class="embed hide-smooth dark">
    <img src="img/Madres.jpg"
        alt="The mothers marching at Plaza de Mayo">
    <figcaption>
        The caption text goes here...
    </figcaption>
</figure>

The resulting captioned image

Usage

Simple, static caption positioned at the bottom of the image...

<figure class="embed">...</figure>

... or at the top.

<figure class="embed-top">...</figure>

Tell the caption to disappear when hovering on the image...

<figure class="embed hide">...</figure>

... or to appear when hovering on the image.

<figure class="embed reveal">...</figure>

Even make the image caption disappear or appear smoothly.

<figure class="embed hide-smooth">...</figure>

A dark or light theme can be used instead of the default...

<figure class="embed dark">...</figure>

... and various font sizes can be applied for larger images.

<figure class="embed large">...</figure>

Combine features to get just the right caption.

<figure class="embed light reveal-smooth large">...</figure>

Just about any combination of these class selectors can be used to get the caption that works for your use case. The following section provides a full listing of the features provided by captionss.

Full Feature Listing

Style:
embed Positions a caption at the bottom.
embed-top Positions a caption at the top.
overlay Overlays the entire image with a caption.
Theme:
neutral Black on light gray (default).
light Gray on lighter gray.
dark White on dark gray.
Font Size:
normal Slightly smaller than page font-size, 0.8em (default).
medium The page's font-size, 1.0em.
large Slightly larger than page font-size, 1.2em.
xlarge Large font-size for large images, 1.5em.
Effect:
hide Hide the caption immediately on mouse hover.
hide-smooth Smoothly hide the caption on mouse hover.
reveal Reveal the caption immediately on mouse hover.
reveal-smooth Smoothly reveal the caption on mouse hover.
